# 波士顿矩阵图可视化工具

一款基于 Streamlit 和 Plotly 开发的交互式波士顿矩阵图（气泡图）生成工具。

## 功能特性

### 数据输入

- **粘贴数据**：支持直接粘贴 CSV 格式或制表符分隔的数据
- **上传文件**：支持上传 CSV、Excel (.xlsx/.xls) 文件
- **自动识别**：自动识别数据列，支持 4 列（无内层气泡）和 5 列数据

### 可视化配置

| 功能   | 说明                       |
| ---- | ------------------------ |
| 图表标题 | 自定义图表标题                  |
| 图表尺寸 | 可调节高度和宽度                 |
| 坐标原点 | 支持自定义 X/Y 轴原点，或使用中位数自动计算 |
| 气泡大小 | 通过乘数和基数调节外层/内层气泡大小       |
| 坐标压缩 | 滑块调节压缩倍数，适用于坐标范围较宽的场景    |

### 文本设置

- 文本大小可调节
- 每个类别可独立设置文本位置（9个方向可选）
- 支持中英文位置标识（居中、左中、右中、上中、左上、右上、下中、左下、右下）

### 悬浮信息

- 自动显示所有指标数据
- 自动计算并显示外层/内层气泡占比（保留1位小数）

### 导出功能

- 一键导出为 HTML 文件，可嵌入网页或分享

## 数据格式

### 5列数据（完整版）

| 列序  | 说明          | 示例         |
| --- | ----------- | ---------- |
| 第1列 | 分类维度（气泡标签）  | 模式         |
| 第2列 | 横坐标（小数形式）   | 广告引入金额\_同比 |
| 第3列 | 纵坐标（小数形式）   | 广告收入\_同比   |
| 第4列 | 外层气泡大小（绝对值） | 广告引入金额     |
| 第5列 | 内层气泡大小（绝对值） | 广告收入       |

### 4列数据（简化版）

支持只输入4列数据，此时内层气泡将使用外层气泡数据。

### 数据示例

```
模式,广告引入金额_同比,广告收入_同比,广告引入金额,广告收入
L1,0.15,0.12,100000,50000
L2,0.08,-0.03,80000,35000
L3,-0.05,0.10,60000,40000
```

## 运行方式

### 安装依赖

```bash
pip install -r requirements.txt
```

### 启动应用

```bash
streamlit run app.py
```

### 访问地址

- 本地：<http://localhost:8501>
- 网络：http\://\[IP地址]:8501

## 技术栈

- **前端框架**：Streamlit
- **可视化库**：Plotly
- **数据处理**：Pandas、NumPy
- **颜色处理**：Matplotlib

- 业务增长分析
- 投资组合评估
- 任意需要四象限分析的业务场景
