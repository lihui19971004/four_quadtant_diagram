# 四象限矩阵图可视化工具 - 功能需求清单

## 一、核心功能

### 1.1 数据输入

- **粘贴数据**：支持直接粘贴 CSV 格式或制表符分隔的数据
- **上传文件**：支持上传 CSV、Excel (.xlsx/.xls) 文件
- **自动识别**：自动识别数据列，支持 4 列（无内层气泡）和 5 列数据

### 1.2 数据列映射

| 列序      | 说明            | 数据类型              |
| ------- | ------------- | ----------------- |
| 第1列     | 分类维度（气泡标签/图例） | 文本                |
| 第2列     | 横坐标           | 小数(百分比) 或 整数(绝对值) |
| 第3列     | 纵坐标           | 小数(百分比) 或 整数(绝对值) |
| 第4列     | 外层气泡大小        | 整数(绝对值)           |
| 第5列（可选） | 内层气泡大小        | 整数(绝对值)           |

### 1.3 可视化配置

#### 文本框输入

- **图表标题**：自定义图表标题
- **图表高度**：默认 800，可调范围 400-2000
- **图表宽度**：默认 1000，可调范围 400-2000
- **坐标原点 X**：小数形式，可自定义
- **坐标原点 Y**：小数形式，可自定义
- **使用自定义原点**：开关选项，默认使用中位数

#### 滑块调节

- **外层气泡乘数**：默认 150，范围 50-300
- **外层气泡基数**：默认 20，范围 5-50
- **内层气泡乘数**：默认 130，范围 50-300
- **内层气泡基数**：默认 15，范围 5-50
- **文本大小**：默认 12，范围 8-24
- **坐标压缩倍数**：默认 0.5，范围 0.1-1.0

### 1.4 文本位置设置

- 自动识别所有类别
- 每个类别可独立设置文本位置
- 9个位置选项（中文标注）：
  - 居中 - middle center
  - 左中 - middle left
  - 右中 - middle right
  - 上中 - top center
  - 左上 - top left
  - 右上 - top right
  - 下中 - bottom center
  - 左下 - bottom left
  - 右下 - bottom right
- 默认位置：middle center

### 1.5 悬浮内容

- 显示分类名称
- 显示横坐标、纵坐标值
- 显示外层气泡大小指标 + 占比（保留1位小数）
- 显示内层气泡大小指标 + 占比（保留1位小数）

### 1.6 数字格式化

- \[>=100000000] 显示为 "X.XX亿"
- \[>=10000] 显示为 "X.X万"
- \[>=100] 显示为整数
- \[<100] 显示为小数
- 自动判断数据类型（百分比 vs 绝对值）

### 1.7 坐标轴处理

- 自动识别坐标数据类型（最大值<=1.0 视为百分比）
- 百分比显示 % 符号，绝对值显示格式化数字
- 坐标压缩算法：越远离中心压缩越多
- 压缩倍数可通过滑块调节

### 1.8 图例设置

- 图例标题固定显示第一列（分类维度）名称
- 颜色方案保持原有风格

### 1.9 导出功能

- 一键导出为 HTML 文件
- 使用 CDN 加载 Plotly.js

### 4.2 核心算法

#### 坐标压缩算法

```python
def transform_coordinate(value, origin, compress_factor=0.5):
    diff = value - origin
    if diff <= 0.5 and diff >= -1:
        return diff
    elif diff > 0.5:
        return 0.5 + (diff - 0.5) * compress_factor
    elif diff < -1:
        return -1 + (diff + 1) * compress_factor
    else:
        return diff
```

#### 数字格式化

```python
def format_number(value):
    if pd.isna(value) or value == 0:
        return "0"
    abs_val = abs(value)
    if abs_val >= 100000000:
        return f"{value/100000000:.2f}亿"
    elif abs_val >= 10000:
        return f"{value/10000:.1f}万"
    elif abs_val >= 100:
        return f"{value:.0f}"
    else:
        return f"{value:.1f}"
```
